How To Use
From the template
1. Download file from download button above.
2. Open folder, replace images to the ones you want.
-- app.js file --
3. Delete lines of the array galleryImages depending on the amount of pictures you want.
4. Edit title & title object for each image.
5. Edit the function setInterval to your liking. It's at 3000 which is 3 seconds.
6. Delete any extra array items on line 21, this is for preloading, and if you don't have 10 images, you won't need those in the array.
In your project
1. Download template
2. Pull in JavaScript file
3. Pull in the CSS file
4. Edit image paths in JavaScript array
5. Edit the CSS to your liking
6. Enjoy!
